# ToDo API (Матрица Эйзенхауэра)

## Описание
Простой REST API для управления задачами по матрице Эйзенхауэра.  
Задачи делятся на 4 квадранта:  
- **Q1**: Важно + Срочно  
- **Q2**: Важно + Не срочно  
- **Q3**: Не важно + Срочно  
- **Q4**: Не важно + Не срочно  

Проект разработан в рамках учебной практики по дисциплине «Разработка кроссплатформенных бизнес-приложений».

## Технологии
- Python 3.11+
- FastAPI 0.110+
- Uvicorn
- Git / GitHub
